Correção do Erro "firebase.database is undefined"

Descrição do Problema

O erro "firebase.database is undefined" geralmente ocorre quando:

  • O script do Firebase Database não foi carregado corretamente
  • A ordem de carregamento dos scripts está incorreta
  • Há conflito entre versões do Firebase
  • O Firebase não foi inicializado antes de usar suas funções

Solução

Para corrigir este problema, precisamos garantir que:

  1. Todos os scripts do Firebase são carregados na ordem correta
  2. O Firebase é inicializado antes de qualquer tentativa de uso
  3. Não há conflito de versões
